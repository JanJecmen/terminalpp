Powerful Cross-Platform Terminal Emulator

Terminal++ (tpp) is a simple, but powerful terminal emulator which provides same features on both Windows, and Linux (MacOS support planned). Although tpp is in alpha stage, it already provides all necessary features to be used as a daily driver.

Features

native support for Windows (DirectWrite) and Linux (X11)
support for modern terminal features (16M colors, mouse, zoom, clipboard, etc.)
fullscreen mode
optional ConPTY bypass for Windows via WSL for better speed and more features
cmd.exe, Powershell and WSL support for Windows
fast (Linux, Windows ConPTY), or really fast (Windows WSL Bypass)
